# site-main-resource

## 项目简介

本仓库用于归档和发布多个独立 HTML 页面。这些页面内容各自独立，不依赖于特定域名或网站环境，适合作为静态资源存放与分发的基础目录。

## 目录说明

```
/
├── index.html          # 仓库入口页（可选）
├── pages/              # 存放独立 HTML 页面
├── assets/             # 公共资源（样式、脚本、图片等）
└── README.md           # 本文件
```

- `pages/` 文件夹：每个子文件夹或文件代表一个独立的 HTML 页面项目，可按需命名。
- `assets/` 文件夹：存放页面共用的样式表、JavaScript 文件、图片等资源，便于统一维护。

## 页面归档说明

所有页面均为纯静态 HTML，可直接通过浏览器打开或部署至任意静态托管服务。页面之间无强关联，可单独更新或删除。建议在每个页面文件内注明其用途或版本信息，以便长期维护。

## 维护说明

- 新增页面：在 `pages/` 下创建新文件夹或文件，并将相关资源放置于 `assets/` 中（如有需要）。
- 更新页面：直接修改对应 HTML 文件，注意保持资源引用路径正确。
- 删除页面：移除对应文件或文件夹，同时清理 `assets/` 中不再使用的资源。
- 欢迎提交 Issue 或 Pull Request 对页面内容或结构提出改进建议。

## 许可

本仓库内容按需授权，具体许可信息请参见各页面文件内的声明。如无特别说明，默认仅供学习与参考使用。
